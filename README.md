# Homebrew Tap for Deluge

This is the official Homebrew tap for [deluge](https://github.com/mstjr/deluge).

## Installation

If not already installed, install homebrew from [https://brew.sh/](https://brew.sh/)

Then, to install `deluge` from this tap, run:

```bash
brew tap mstjr/deluge
brew install mstjr/deluge/deluge
```
